AI Financial Risk & Spending Intelligence System

Overview

This project is an AI-powered financial analytics system that analyzes spending behavior, predicts future expenses, detects anomalies, and evaluates financial risk using Machine Learning techniques.
The system is built using Python and deployed as an interactive dashboard using Streamlit.

Features

* Financial data analysis and visualization
* Expense forecasting using Linear Regression
* Financial risk evaluation based on savings ratio
* Anomaly detection using Isolation Forest
* Interactive dashboard using Streamlit

Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* Plotly

How to Run the Project

1. Clone the repository

```
git clone https://github.com/PilliSnigdha/AI-Financial-Risk-System.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the Streamlit app

```
streamlit run app.py
```
